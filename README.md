# domino-tracker

PWA para apuntes de dominó — offline, sin login, todo en localStorage.

## Usar en el teléfono

1. Abre: **https://dilsonzapata.github.io/domino-tracker/**
2. En Safari (iOS): Compartir → **Añadir a pantalla de inicio**
3. En Chrome (Android): Menú → **Instalar app**

## Archivos

- `index.html` — entrada para GitHub Pages
- `domino-tracker.html` — misma app (copia local)

## Desarrollo local

```bash
npx serve .
```
